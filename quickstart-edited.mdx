Augment is the developer AI for teams that deeply understands your codebase and how you build software. Use Augment understand code, debug issues, and ship faster.

This topic shows you how to install Augment, sync your repository, and start coding.

## Sign in

* Visual Studio Code and JetBrains IDEs: Select the Augment icon in your IDE side panel, or press <Keyboard shortcut="Cmd/Ctrl + L" /> to open the Augment panel. Select **Sign in to Augment** to sign in.
* Auggie CLI: Run `auggie login` command in your terminal to sign in.

For more information on how to sign in to or out of Augment, see the [Sign in and out](/setup-augment/sign-in) topic.

## Index your workspace

When you first use Augment, it's recommended to index your workspace. Indexing your workspace allows Augment to tailor suggestions to your codebase.

* Visual Studio Code and JetBrains IDEs: Open the Augment panel. The panel prompts you to index your codebase.
* Auggie CLI: Auggie CLI prompts you to index your codebase when you launch Auggie. (Launch Auggie by typing `auggie` in your terminal.)

For more information about syncing your repository, see the [Index your workspace](https://docs.augmentcode.com/setup-augment/workspace-indexing) topic.

After you sign in and sync your repository, you're ready to start coding with Augment. The following features are available in the Visual Studio Code extension.

### Agent

Augment Agent helps you complete end-to-end software development tasks like writing code, debugging, and testing. To access Agent, select the `Agent` tab in the Augment panel and use natural language to describe your tasks.

For more information, see [Using Agent](https://docs.augmentcode.com/using-augment/agent).

### Next Edit

Next Edit suggests changes based on your context and recent changes. Next Edit lets you quickly make changes to your code, and still have full control to accept or reject suggested changes. When you're viewing code, jump to suggested changes from Next Edit by pressing `Cmd/Ctrl + ;`.

For more information, see [Next Edit](https://docs.augmentcode.com/using-augment/next-edit).

### Instructions

Instructions is a feature that edits a specific code block or inserts code in a specific location. To use Instructions, first highlight the code you want to edit. Then press `Cmd/Ctrl + I` or place your cursor on the location you want to add new code.

For more information, see [Instructions](https://docs.augmentcode.com/using-augment/instructions).

For more information, see [Completions](https://docs.augmentcode.com/using-augment/completions).

## Code with Auggie CLI

Type `auggie` in your terminal to begin using Auggie. Use Auggie to assist with coding, get feedback on PRs, and more. For more information how to use Auggie, see [Introducing Auggie CLI](https://docs.augmentcode.com/cli/overview).
